
# LaTienditaDeGilberto  - Curso Coderhouse ReactJS


Proyecto realizado con ReactJS para el curso de Coderhouse. 

eCommerce 
Tecnologias usadas:
ReactJS - Vite
CSS
JavaScript

Proyecto que incluye el uso de React-Router-Dom, UseContext, UseState, FireStore como Base de Datos.

Familia de Iconos: Phosphor Icons, para un mejor estilo visual. 


Para realizar este proyecto, trate de usar un enfoque de un eCommerce que pudiese tener diferentes categorias en cuanto a los productos mostrados, para asi poner en pratica la navegaci√≥n con react-router-dom. Queria una pagina que fuese visualmente atractiva, por lo que trate de usar un buen contraste en los colores, al igual que bordes e imagenes. 

Al mismo tiempo, usar dependencias como useState para poder hacer estos cambios en el DOM posibles.

Usamos firebase para asi poder generar una base de datos sin necesidad del uso de backend, algo que nos permitira tener ese almacenamiento de informacion, por ejemplo, para las ordenes de compra, productos. El uso del mismo tambien nos ayudara a que podamos hacer muchos cambios sin necesidad de tocar o modificar el codigo de React.
